jQuery的内容

jQuery

入口函数的两种写法

jquery的入口函数：作用等待页面的加载完成。不会等待图片和文件的加载

第一种：$(document).ready(function(){

});

第二种：$(function(){

});

jQuery名称冲突

jQuery 使用 $ 符号作为 jQuery 的简介方式。

某些其他 JavaScript 库中的函数（比如 Prototype）同样使用 $ 符号。

jQuery 使用名为 noConflict() 的方法来解决该问题。

var jq=jQuery.noConflict()，帮助您使用自己的名称（比如 jq）来代替 $ 符号

jquery对象和DOM对象的相互转换

var $li = $(“li”);

//第一种方法（推荐使用）

$li[0]

//第二种方法

$li.get(0)

DOM对象转jquery对象

var $obj = $(domobj);

选择器

过滤选择器

$(“li:eq(2)”).css(‘color’,’red’) 描述：获取到的li元素中，选择索引号为2的元素，索引号index从0开始

$(“li:odd”).css(‘color’,’red’) 描述：获取到的li元素中，选择索引号为奇数的元素

$(“li:even”).css(“color”,”red”) 描述：获取到的li元素中，选择索引号为偶数的元素

筛选选择器

children（）子类选择器

find（）后代选择器

siblings（）查找兄弟节点，不包括自己本身

parent（）查找父亲

eq（index）

next（）找下一个兄弟

prev（）找上一个兄弟

index（）获取当前元素在所有兄弟里面的索引

元素选择器

$(“p.intro”) 选取所有 class=”intro” 的

元素。

$(“p#demo”) 选取所有 id=”demo” 的

元素

jquery事件

mouseenter 移入事件

mouseleave 移出事件

stop（）停止正在执行的动画

slideDown() 往下滚动出现

slideUp() 往上滚动出现

fadeIn（）淡入

fadeOut（）淡出

find（）方法获得当前元素集合中每一个元素的后代，通过选择器、jQuery对象或元素来筛选

end（）方法结束当前链条中的最近的筛选操作，并将匹配元素集还原为之前的状态

html（）方法返回或者这是备选元素的内容（innerHTML）

text（）把标签当做文本，IE提出来的，所以低版本的火狐浏览器有兼容性的问题。

val（）方法返回或设置被选元素的值。

get（）方法获取由选择器指定的DOM元素。

eq（）方法将匹配元素集缩减值指定index上的一个。

focus（）方法：当元素获得焦点时，发生focus事件；

当通过鼠标点击选中元素或通过tab键定位到元素时，该元素就会获得焦点。

blur（）方法：当元素失去焦点时发生blur事件

trigger（）方法触发被选元素的指定事件类型。

获取可视区的宽度和高度：

$(window).width();

$(window).height();

bind：绑定

​ $(“p”).bind(‘click’,fn);

​ 清除所有类型的点击事件

​ $(“p”).unbind(‘click’,fn)

empty（）清除内容

remove（）删除所有的内容，包括自己

disabled 属性规定应该一组表单元素,被禁用的表单元素，不可用也能点击

selected是select的默认选中要求写在option标签上

trigger（）；触发事件

toggle（）；切换

e.preventDefault（）阻止浏览器的默认行为

e.stopPropagation（）阻止事件冒泡的行为

return false 既可以阻止浏览器的默认行为，还可以阻止事件冒泡行为

each（）遍历jquery对象

prevAll（）获得当前匹配元素集合中每个元素的前面的同胞元素，使用选择器进行筛选是可以选的。

nextAll（）获得匹配元素集合中每个元素的所有跟随的同胞元素，由选择器筛选是可选的。

clone（）克隆节点；true：赋值事件

on注册事件的两种方式：

普通的事件（给自己的执行事件）；

委托事件（给父元素注册，给子元素执行）；

​ 第一个参数：事件的类型；

​ 第二个参数：选择器 如果不写选择器，说明这个事件是给自己，注册事件由自己执行。

​ 第三个参数：data 可以存一个值，在注册事件的时候，可以传递一个值，这个值保存到event.data属性中

​ 第四个参数：function
